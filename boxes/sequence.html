Sequence & Sets

Sequence

A sequence is a list of things in some arbitrary order. For example, { 5,4,3,2,1 }\set{5, 4, 3, 2, 1}{5,4,3,2,1} is a sequence of numbers starting from 5 ascending down. In programming, you may see these as arrays that start at 0. In math, sequences start at 1.

let sequence = [5, 4, 3, 2, 1];
{ 3,3,1 }\set{ 3, 3, 1 }{3,3,1}

Sets

You may also have heard of sets. These exist in math as well, and like in many languages, elements can only be repeated once, but the order doesn’t matter. The sequence { 1,2,3,3 }\set{1, 2, 3, 3}{1,2,3,3} would be a set of { 1,2,3 }\set{1, 2, 3}{1,2,3}. You can try sets in JavaScript using the using the Set object.

let set = new Set([1, 1, 2, 3]);
{ 1,2,2,3 }\set{ 1, 2, \textcolor{red}{2}, 3 }{1,2,2,3}

Summation

The following is the syntax for a summation, one of the many commonly used mathematical symbols. This is exhibited with a for loop:

let sum = 0;
let c = ...;

for (let n = a; n <= b; n++) {
  sum += c;
}
